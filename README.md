# Smart Translator

## Overview
**Smart Translator** is a Python desktop application built with **Tkinter** that allows users to translate text between multiple languages in real-time. It uses the **Deep Translator** library (Google Translate API) to perform translations.

## Features
- Translate text between English, Hindi, Nepali, French, and German.
- User-friendly GUI with Tkinter.
- Real-time translation on button click or pressing Enter.
- Displays both source and translated text.

## How It Works
1. User selects the source and target languages from dropdown menus.
2. User enters text in the "Source Text" box.
3. Press **Translate** or Enter to see the translated text in the "Translated Text" box.
4. The app uses **GoogleTranslator** from `deep_translator` to perform translations.
